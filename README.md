# Логгер Действий Discord (DSharpPlus)

## Описание

Этот бот разработан на основе библиотеки DSharpPlus и предназначен для логирования действий пользователей на сервере Discord.

### События, логируемые ботом:
- `GuildBanAdded`: Событие вызывается при добавлении пользователя в список забаненных на сервере.
- `GuildBanRemoved`: Событие вызывается при удалении пользователя из списка забаненных на сервере.
- `GuildMemberAdded`: Событие вызывается при добавлении нового участника на сервер.
- `GuildMemberRemoved`: Событие вызывается при удалении участника с сервера.
- `MessageDeleted`: Событие вызывается при удалении сообщения на сервере.
- `MessageUpdated`: Событие вызывается при обновлении сообщения на сервере.

--------------------------

## Description

This bot is developed based on the DSharpPlus library and is designed to log user actions on a Discord server. 

### Events Logged by the Bot:
- `GuildBanAdded`: Triggered when a user is added to the ban list on the server.
- `GuildBanRemoved`: Triggered when a user is removed from the ban list on the server.
- `GuildMemberAdded`: Triggered when a new member is added to the server.
- `GuildMemberRemoved`: Triggered when a member is removed from the server.
- `MessageDeleted`: Triggered when a message is deleted on the server.
- `MessageUpdated`: Triggered when a message is updated on the server.
